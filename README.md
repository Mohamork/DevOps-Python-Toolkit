# DevOps-Python-Toolkit
A collection of small Python projects focused on cloud, automation and networking tools.


## 🛜 Networking Tools
 - Portscan simulator - Check if a given port is open on remote hosts.
 - Port status checker - Categorize port types (critical or non-critical)

## 🌫️ Cloud Tools
 - Cloud VM inventory tracker - Track VM state and count running vs stopped vms.
